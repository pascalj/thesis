This is an overengineered thesis template.

It uses:

\begin{itemize}
\tightlist
\item
  \href{https://nixos.org/}{Nix}
\item
  \href{https://pandoc.org/}{Pandoc}
\item
  \LaTeX
\item
  \href{https://graphviz.org/}{dot}
\item
  \href{https://inkscape.org}{Inkscape}
\item
  \href{https://www.python.org/}{Python}
\end{itemize}

\begin{figure}
\hypertarget{fig:png}{%

\caption{A graph from a PNG file}\label{fig:png}
}
\end{figure}

It can embed PNG images (see \cref{fig:png}), SVG images via Inkscape (see \cref{fig:svg}) or DOT graphs (see \cref{fig:dot}).
You can also cite references like~\autocite{moore_cramming_1965}, use inline math like \(\frac{1}{n}\) or display math:

\begin{align}
  0 + 1 + 2 + 3 + 4 + \dotsb + n = 
  \sum_{k=0}^n k = \frac{n(n+1)}{2} = \frac{n^2+n}{2}
\end{align}

Just include it like seen in \cref{lst:display}.
